# 🛡️ Papy3D NoticeShield

A modern and powerful WordPress admin notice manager designed to keep your dashboard clean, readable and under control — without hiding important information blindly.

Unlike basic “hide admin notices” plugins, NoticeShield gives you full visibility and reversible control over every captured notice.

---

## ✨ Features

### 🎯 Smart Notice Management

* Allow / Block / Ask Again decisions
* Reversible actions at any time
* Per-notice filtering
* Per-source filtering

### 🧠 Learning Mode

New notices stay in a safe “Ask Again” state until an administrator explicitly decides what to do.

### 🛡️ Safe Core Mode

Protects important WordPress core workflow notices from accidental hiding.

### 📦 Source Control

See which plugins generate notices and manage them globally.

### 🕘 Local History

Keep a local history of captured notices and admin decisions.

### 🔄 Import / Export

Easily transfer your rules and settings between websites using JSON exports.

### ⏸️ Temporary Pause

Pause NoticeShield for one hour during troubleshooting or maintenance.

# 🚀 Why NoticeShield?

Papy3D NoticeShield was built with one goal:

> Keep the WordPress admin clean without losing visibility or control.

Many notice hider plugins simply remove everything.
NoticeShield takes a safer and more professional approach.

# 📥 Installation

1. Upload the plugin to `/wp-content/plugins/`
2. Activate the plugin through the WordPress admin
3. Open **Tools → NoticeShield**

---

# 🧩 Plugin Philosophy

NoticeShield is intentionally lightweight and focused.

It does not:

* inject ads,
* promote premium upgrades,
* send data externally,
* modify unrelated admin areas.

It simply helps administrators regain control over noisy dashboards.
